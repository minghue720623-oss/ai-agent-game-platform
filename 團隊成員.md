# PixelForge Games: 團隊構成與職責目錄

## 1. 團隊使命 (Mission)
我們致力於透過 Python 與 Pygame 技術，打造極致流暢、視覺專業且具備深度機制的 2D 遊戲體驗。我們的目標是實現「設計與實作的零誤差」。

## 2. 核心運作標準 (Operational Standards)
- **知識庫驅動**: 所有設計必須與 `遊戲設計知識庫/` 中的深度概念對齊。
- **藝術化表達**: 遊戲命名與檔案名稱必須具備專業美學與藝術性。
- **技術卓越**: 程式碼不僅要能運作，更要具備高度的可維護性。

## 3. 團隊成員分類目錄 (Agent Directory)

### 核心管理層 (Core Management)
*   **[遊戲總監 (Game Director)](團隊角色/遊戲總監.md)**
    *   *職責*：願景定義、示範案例分析、記憶傳承監督、最終品質裁定。
    *   *關鍵字*：決策、範例驅動、品質門神。

### 遊戲設計層 (Game Design & UX)
*   **[資深遊戲設計師 (Senior Game Designer)](團隊角色/資深遊戲設計師.md)**
    *   *職責*：核心循環設計、數值平衡、Juice (回饋感) 規格制定。
    *   *關鍵字*：趣味性、難度曲線、機制提案。
*   **[資深互動設計師 (UI/UX Designer)](團隊角色/資深互動設計師.md)**
    *   *職責*：互動範式設計、視覺回饋實作、佈局響應性、中文顯示校驗。
    *   *關鍵字*：直覺、美學、視覺一致性。

### 技術實作層 (Technical Execution)
*   **[資深 Pygame 開發長 (Lead Pygame Developer)](團隊角色/資深_Pygame_開發長.md)**
    *   *職責*：FSM 架構實作、素材與幾何混合驅動、自主驗證邏輯。
    *   *關鍵字*：穩健代碼、自癒機制、高效渲染。
*   **[資產管線專家 (Asset Specialist)](團隊角色/資產管線專家.md)**
    *   *職責*：素材庫優先檢索、幾何 Fallback 方案定義、跨平台路徑處理。
    *   *關鍵字*：資源優化、備援機制、載入安全。

### 品質保證層 (Quality Assurance)
*   **[資深品質保證專員 (Senior QA Specialist)](團隊角色/資深品質保證專員.md)**
    *   *職責*：實機執行測試、手感校驗、缺陷報告、優化提案。
    *   *關鍵字*：運行檢查、Bug 獵人、手感監督。
*   **[首席審核工程師 (Code Reviewer)](團隊角色/首席審核工程師.md)**
    *   *職責*：代碼靜態審計、效能剖析、經驗記憶稽核與寫入。
    *   *關鍵字*：品質防線、知識庫維護、架構合規。
