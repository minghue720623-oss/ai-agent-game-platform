# PixelForge Games - 多樣性遊戲開發索引

本專案旨在提供全類型遊戲開發支援，透過雙軌互動機制為不同需求的使用者提供開發協助。

---

## 1. 雙軌式設計啟動流程 (Two-Track Design Initiation)

AI 團隊根據使用者的設計意圖，採用不同的協助策略：

### 軌道一：創意催化 (Creative Facilitation) - 針對有想法的用戶
*   **AI 角色**：技術轉譯與可行性分析者。
*   **目標**：將模糊的創意快速轉化為具體的遊戲開發規格與開發時程。
*   **互動模式**：直接鎖定核心機制，進行規格討論與技術可行性驗證。

### 軌道二：啟發與探索 (Inspiration & Exploration) - 針對沒想法的用戶
*   **AI 角色**：類型引導與風格發想師。
*   **目標**：透過問卷、類型對比與範例展示，輔助使用者定義遊戲目標。
*   **互動模式**：發散式思維引導，從風格、機制與題材中篩選出潛力方向。

---

## 2. 模組化知識庫與素材參考
*   **通用設計資源**：`遊戲設計知識庫/` 資料夾中的內容為各類型通用的參考資料。
*   **示範案例**：現有的遊戲示範（如《霓虹食界》）僅作為特定類型實現的參考案例，開發時請根據目標遊戲類型動態調用參考，而非強制採用該風格。

---

## 3. 全球發布體系
- **發布管理專家**：負責將作品發布至使用者的個人 GitHub 帳號。
- **發布流程**：
  1. 自動偵測當前環境登入的 GitHub 帳號。
  2. 詢問使用者上傳意願。
  3. 根據遊戲類型在該帳號下建立獨立儲存庫。
  4. 推送代碼並更新遠端連結位址。
