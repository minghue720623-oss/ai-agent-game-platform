# Neon Glutton: Infinite Link (霓虹食界：無限鏈結)

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Pygame](https://img.shields.io/badge/Pygame-Latest-green?style=for-the-badge)

這是一款由 **PixelForge AI Team** 開發的霓虹風格貪食蛇遊戲。玩家在電子空間中操作能量鏈結，透過吞噬能量塊來維持系統穩定。

## 遊戲特色
- **視覺風格**：深紫色電子背景搭配綠色霓虹蛇身。
- **操作簡單**：使用方向鍵控制，目標是獲得最高能量點。
- **即時回饋**：包含分數紀錄與系統崩潰（遊戲結束）重啟機制。

## 安裝與運行
1. 確保已安裝 Python 3.x。
2. 安裝必要的套件：
   ```bash
   pip install pygame
   ```
3. 執行遊戲：
   ```bash
   python 霓虹食界_無限鏈結.py
   ```
